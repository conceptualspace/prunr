## About
- Prunr is an Electron duplicate file scanner and utility.
- Electron forge + webpack for builds
- React + Mantine on the frontend
- See package.json for more detail

## Conventions
- Prefer functional code style
- Do not remove or edit existing comments unless they would be invalid
- Do not add verbose comments to the code
- Challenge the user when confident about an opinion

## Run App
Run with `npm start` to launch.
